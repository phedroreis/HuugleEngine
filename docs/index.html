Instalação da Cópia Estática do Fórum Clube Cético

Descompacte o arquivo baixado.

No sistema Windows copie a pasta clubecetico.org para qualquer local. No Ubuntu copie para a pasta de sistema /opt.

Na pasta clubecetico.org basta clicar sobre o arquivo index.html (ou index-2.html) para reproduzir a experiência de navegar pelo antigo fórum.

Instalação dos Arquivos de Índice da Ferramenta de Busca Huugle

Descompactando o arquivo baixado obtém-se a pasta database, que contém principalmente os arquivos de índice necessários à ferramenta de busca.

É necessário baixar este arquivo se você pretende utilizar a ferramenta de busca Huugle. Nesse caso o executável do programa deve ser copiado para uma pasta de nome Huugle que deve ser criada dentro de uma outra pasta nomeada como clubecetico.org. Ficando: clubecetico.org/Huugle.

Ainda, a pasta database deve ser copiada para esta mesma pasta Huugle para que o programa possa acessar sua base de dados e realizar buscas de títulos de tópicos e posts a partir das palavras chaves fornecidas pelo usuário.

Mas se você baixou também, ou pretende baixar, a cópia estática do fórum, então uma pasta de nome clubecetico.org já existe ou vai existir em seu sistema. É a pasta com as páginas do fórum e demais arquivos necessários à sua exibição. Logo, a pasta Huugle deve ser criada dentro desta pasta que você baixou (ou a ser baixada) compactada no arquivo clubecetico.org.7z. Isso vai permitir que os links criados pelo programa (os resultados de uma pesquisa) apontem para páginas na sua cópia local e você possa abrir estas páginas no seu navegador.

Em outras palavras, os links para páginas da cópia local esperam que estes arquivos HTML estejam em uma pasta de nome clubecetico.org e que a própria ferramenta de busca Huugle também esteja gravada em uma pasta dentro da pasta clubecetico.org.

Em sistemas Windows tanto o programa quanto a navegação pelo simulacro estático do fórum irão funcionar se a pasta clubecetico.org for instalada em qualquer local. Em sistemas Debian-Ubuntu deve ser instalada dentro da pasta de sistema /opt.

Obter e Compilar o Código Fonte da Ferramenta de Busca Huugle

Não é fornecido nenhum instalador, para instalar o programa em seu sistema você deve proceder a instalação manual dos arquivos baixados conforme as instruções nesta página.

Para compilar os arquivos obtidos é recomendado Oracle JDK14 ou superior.


Para Windows (qualquer versão)

Descompacte a pasta src do arquivo baixado no link.

Entre nessa pasta e execute o arquivo compileHuugle.bat contido nela. Isso irá gerar o HuugleEngine.jar, que é o executável do programa.

Talvez o path no seu sistema não esteja configurado para localizar o seu pacote de instalação do JDK (Java Development Kit). O que é necessário para que o script compileHuugle.bat possa chamar o compilador javac. Se for preciso, localize na pasta C:\Arquivos de Programas o caminho completo até o diretório bin na pasta onde o JDK foi instalado e digite no terminal o comando path semelhante a esse:

path C:\Arquivos de Programas\java\jdk8.0.14\bin

Observando que no seu sistema o caminho até a pasta bin pode ser diferente. Além disso, note que para compilar o projeto não basta ter o Java instalado. É necessário o JDK.


O executável precisa de seus arquivos de índice que estão em uma pasta de nome database e que pode ser baixada aqui. Crie uma pasta de nome Huugle e copie a pasta database e o arquivo HuugleEngine.jar para ela.

Apenas isso é necesário para executar a ferramenta Huugle em sistemas Windows. Porém a ferramenta, a cada solicitação de busca, abre uma página no navegador com links para os resultados encontrados. Você pode selecionar se estes links irão apontar para páginas da cópia estática hospedada na web , ou para as páginas da sua própria cópia estática no seu sistema de arquivos local. Caso tenha baixado e instalado a cópia estática do fórum em sua máquina.

No entanto para solicitar ao Huugle que aponte os resultados de uma busca para arquivos instalados em seu próprio sistema, é necessário que esta pasta Huugle tenha sido copiada para a pasta clubecetico.org, que você obtém ao descompactar este arquivo.


Para Ubuntu

Em primeiro lugar é aconselhável que você tenha Oracle JDK14 ou superior para compilar e executar o programa. Por default sistemas Debian instalam o OpenJDK, o que pode causar algum problema de incompatibilidade.

Para substitui-lo pelo Oracle JDK siga as instruções abaixo.

Desinstale o Open JDK com o comando:

$ sudo apt-get remove --purge openjdk-*

Então instale a última versão do Oracle JDK que hoje, 20 de agosto de 2020, é o 14. Utilize os seguintes comandos:

$ sudo add-apt-repository ppa:linuxuprising/java
$ sudo apt update
$ sudo apt install oracle-java14-installer

Ao final o terminal irá mostrar o contrato de licença da Oracle. Role a tela até o fim e tecle ESC. Surge a opção de aceitar e você seleciona sim e tecla ENTER.

Agora você já pode executar o script compileHuugle.sh, um aquivo que está dentro da pasta src contendo os fontes do programa para Debian-Ubuntu e que você baixou neste link.

Antes lembre-se de acessar as propriedades deste arquivo e marcar a opção "Tornar o arquivo executável", caso contrário o Linux não executará o script.

$ ./compileHuugle.sh

Isto irá gerar o executável HuugleEngine.jar mas, novamente, no Linux para que um arquivo possa executar como um programa é preciso alterar seus atributos e conceder essa permissão. O que pode ser feito na janela de Propriedades do Arquivo.

Para instalar o programa no seu sistema, crie dentro da pasta /opt uma pasta de nome /opt/clubecetico.org/Huugle, para onde deve ser copiado este arquivo. Juntamente com a pasta database que contém os arquivos de índice da ferramenta e pode ser baixada neste link

Mas lembre-se que por padrão /opt tem acesso de escrita apenas para usuário root, portanto, para copiar as pastas e arquivos abra um terminal e digite:

$ sudo -i
$ nautilus

Irá abrir o gerenciador de arquivos no modo root de maneira a ser possível fazer as alterações necessárias.

Ainda, se desejar instalar a cópia estática do Fórum Clube Cético fornecida para download também nessa página, note que ao descompactar este arquivo você obtém a pasta clubecetico.org com o simulacro navegável do fórum.

Nesse caso, se você já tem ou vai obter esta cópia , então apenas copie esta pasta para /opt e depois crie a pasta Huugle dentro dela. Para onde devem ser copiados o executável HuugleEngine.jar gerado pela compilação dos fontes e também a pasta dabatase, com a base de dados que este executável necessita para realizar as pesquisas no acervo do fórum.
